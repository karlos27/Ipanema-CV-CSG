Aquest és el plugin del meu curriculum vitae. 
 
== Description == 
 
He creat aquest plugin en format curriculum per a poder compartir les meves dades personals i història laboral i així poder mostrar les meves habilitats com a desenvolupador web.

 
== Installation == 
 
1. Download the plugin 
1. Upload the ipanema-cv-csg folder to your site's wp-content/plugins directory 
1. Activate the plugin in the WordPress Admin Panel
1. Watch my CV in the Curriculum vitae level in the admin menu
1. Enjoy the music in the help tabs!
1. Thank you!
